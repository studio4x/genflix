# Sprint 2 - Conteudo Admin (Obrigatorios)

## Escopo implementado

1. CRUD de cursos.
2. CRUD de modulos com ordenacao manual (subir/descer).
3. CRUD de aulas com suporte a URL do YouTube.
4. Upload e listagem de materiais complementares por aula.

## Rotas admin adicionadas

1. `/admin/cursos`
2. `/admin/cursos/:courseId/modulos`
3. `/admin/modulos/:moduleId/aulas`
4. `/admin/aulas/:lessonId/materiais`

## Fora do escopo desta sprint

1. Liberacao de cursos para aluno.
2. Progresso por aula/modulo.
3. Avaliacoes.
4. Relatorios.
