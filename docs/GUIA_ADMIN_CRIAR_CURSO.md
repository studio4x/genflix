# Guia rápido: como criar um curso

Este guia mostra o caminho mais simples para o admin criar um curso no Genflix.

## 1. Acesse o painel de admin

1. Entre no site com sua conta de administrador.
2. No menu principal, abra a área de `Cursos`.

## 2. Crie o curso

1. Clique em `Criar Curso Agora`.
2. Preencha os dados básicos do curso:
   - nome do curso;
   - descrição;
   - imagem de capa;
   - preço, se o curso for pago;
   - status inicial, se quiser deixar como rascunho.
3. Salve para abrir o construtor do curso.

## 3. Monte a estrutura

Dentro do construtor, organize o conteúdo nesta ordem:

1. Crie os módulos do curso.
2. Adicione as aulas dentro de cada módulo.
3. Se necessário, adicione quizzes ou avaliações.
4. Revise a ordem do conteúdo para deixar a trilha de aprendizado clara.

## 4. Ajuste a página pública

1. Abra a aba da página pública do curso.
2. Verifique título, descrição, imagem e chamada principal.
3. Confirme se o curso aparece com a apresentação correta para o aluno.

## 5. Configure as regras do curso

1. Abra as configurações do curso.
2. Confira:
   - liberação das aulas;
   - regras de acesso;
   - avaliações habilitadas;
   - publicação final.
3. Salve as alterações.

## 6. Publique

1. Revise o curso completo.
2. Quando estiver tudo certo, publique.
3. Teste o acesso como aluno para confirmar que o curso abre corretamente.

## Dicas rápidas

- Se o curso ainda não estiver pronto, mantenha como rascunho.
- Sempre teste a versão publicada antes de divulgar.
- Se houver módulos ou aulas faltando, complete a estrutura antes da publicação.
